# DragonSlayer v. 2
DragonSlayer is my first CRUD project made in python with Flask and SQLAlchemy.

DragonSlayer is a ticket manager with role and permissions system.

<img width="959" height="502" alt="Captura de pantalla 2026-07-07 151607" src="https://github.com/user-attachments/assets/4101f821-e36c-48f9-bd97-0dd43797899e" />

## what can DragonSlayer do?
### User system
DragonSlayer has a login system with user that admins and the owner can create and edit.

<img width="959" height="499" alt="Captura de pantalla 2026-07-07 152452" src="https://github.com/user-attachments/assets/5e45f42a-54ff-4c2c-a3b9-300c974bc454" />

### Ticket manager
DragonSlayer can create, edit and conclude tickets with titles, descriptions, priority and status levels.

<img width="959" height="500" alt="Captura de pantalla 2026-07-07 152537" src="https://github.com/user-attachments/assets/2ff2b243-2488-4fc5-97d7-ff8bf2127110" />

### Role system
DragonSlayer has 4 roles with it's self permissions.
* Owner
* Administrator
* Default
* Read only
